
# VoyceMe

## Project Home Test

* Front-end: Next.js / Styled-components / Apollo-client
* Back-end: Node.js / Graphql / Express
* Database: MongoDB using Atlas




## Install and run BACKEND

Install with yarn this project

```bash
  cd api-admin
  yarn
  yarn start:dev
```
    
## Install and run FRONTEND

Install with yarn this project

```bash
  cd admin
  yarn
  yarn dev
```
